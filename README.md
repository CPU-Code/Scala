```scala

}
```

输入 1.var -> 回车

```scala

```


快速打印： .sout

输入 1.sout->回车

```scala

```



快速生成 for 循环： 遍历对象.for


输入 1 to 3.for

```scala
for (elem <- 1 to 3) {
}
```
