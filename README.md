# [IDEA 快捷键]



快速生成程序入口： main

输入 main -> 回车

```scala
def main(args: Array[String]): Unit = {
}
```

自动补全变量： .var

输入 1.var -> 回车

```scala
val i: Int = 2
```


快速打印： .sout

输入 1.sout->回车

```scala
println(1)
```



快速生成 for 循环： 遍历对象.for


输入 1 to 3.for

```scala
for (elem <- 1 to 3) {
}
```
